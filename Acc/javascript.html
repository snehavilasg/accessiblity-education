JavaScript Basics for Learners

Introduction to JavaScript

JavaScript is a programming language used to make web pages interactive. It allows you to manipulate the content of a web page, respond to user inputs, and perform complex actions. This tutorial will cover some basic JavaScript concepts.

Variables

Variables are used to store data values. You can declare a variable using the let, const, or var keywords.

/* Declaring variables */
let name = "John";       // String
const age = 25;          // Number
var isStudent = true;    // Boolean

In the above example, name is a string, age is a number, and isStudent is a boolean.

Functions

Functions are blocks of code designed to perform a particular task. You can define a function using the function keyword.

/* Defining a function */
function greet() {
    console.log("Hello, World!");
}

/* Calling the function */
greet();

In this example, we define a function called greet that logs "Hello, World!" to the console when called.

Events

JavaScript can be used to interact with users through events, such as clicking a button. You can use event listeners to trigger functions when an event occurs.

When the button is clicked, the JavaScript function shows an alert with the message "Button clicked!".

Loops

Loops allow you to repeat a block of code multiple times. The most common types of loops in JavaScript are the for loop and the while loop.

For Loop Example:

for (let i = 0; i < 5; i++) {
    console.log(i);  // Prints 0, 1, 2, 3, 4
}

While Loop Example:

let i = 0;
while (i < 5) {
    console.log(i);  // Prints 0, 1, 2, 3, 4
    i++;
}

The for loop repeats a block of code a specific number of times, while the while loop continues as long as the condition is true.

Conditionals

Conditionals allow you to execute different code based on different conditions. You can use if, else if, and else to make decisions.

/* Using if-else statements */
let age = 20;
if (age >= 18) {
    console.log("You are an adult.");
} else {
    console.log("You are a minor.");
}

In this example, the program checks if the age is greater than or equal to 18 and outputs the appropriate message.
