Basic HTML Tags for Learners

Introduction to HTML

HTML (HyperText Markup Language) is the standard language for creating webpages. HTML uses a variety of "tags" to structure content, such as headings, paragraphs, images, links, and lists. In this guide, we will cover some of the most commonly used HTML tags with examples.

Headings

Headings are used to define the structure of a page. HTML has six levels of headings, from <h1> (the largest) to <h6> (the smallest).

<h1>This is a Heading 1</h1>
<h2>This is a Heading 2</h2>
<h3>This is a Heading 3</h3>

Example:

This is a Heading 1


This is a Heading 2


This is a Heading 3

Paragraphs

Paragraphs are used to define blocks of text. In HTML, the <p> tag is used to create a paragraph.

<p>This is a paragraph of text.</p>

Example:

This is a paragraph of text.

Links

Links are used to connect one page to another or to external resources. The <a> tag is used to define a link. The href attribute specifies the destination of the link.

<a href="https://www.example.com">Click here to visit Example</a>

Example:

Click here to visit Example

Images

Images can be added using the <img> tag. The src attribute specifies the path to the image, and the alt attribute provides alternative text if the image is not loaded.

<img src="image.jpg" alt="Description of image">

Lists

There are two types of lists in HTML: ordered lists and unordered lists.

Unordered List

An unordered list is a list of items that are not in any specific order. The <ul> tag is used to create an unordered list, and <li> tags are used for each list item.

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

Ordered List

An ordered list is a list of items in a specific order. The <ol> tag is used to create an ordered list, and <li> tags are used for each list item.

<ol>
  <li>First Item</li>
  <li>Second Item</li>
  <li>Third Item</li>
</ol>

Tables

Tables are used to display data in rows and columns. The <table> tag is used to create a table, <tr> for rows, <th> for table headers, and <td> for table data.

<table>
  <tr>
    <th>Header 1</th>
    <th>Header 2</th>
  </tr>
  <tr>
    <td>Data 1</td>
    <td>Data 2</td>
  </tr>
</table>
