C# Basics for Learners

What is C#?

C# (pronounced C-sharp) is a modern, object-oriented programming language developed by Microsoft. It is primarily used for developing Windows applications, web applications, and games (using the Unity game engine).

C# Syntax

C# syntax is similar to other C-based languages such as C++, Java, and JavaScript. Here's an example of the basic structure of a C# program:

using System;

class Program
{
    static void Main(string[] args)
    {
        Console.WriteLine("Hello, World!");
    }
}

This simple program outputs "Hello, World!" to the console.

C# Variables and Data Types

In C#, variables are used to store data, and each variable has a data type (such as int, string, float, etc.).

using System;

class Program
{
    static void Main(string[] args)
    {
        int age = 25;           // Integer
        double height = 5.9;    // Double (floating-point)
        string name = "John";   // String
        bool isStudent = true;  // Boolean

        Console.WriteLine("Name: " + name);
        Console.WriteLine("Age: " + age);
        Console.WriteLine("Height: " + height);
        Console.WriteLine("Student: " + isStudent);
    }
}

This program demonstrates different types of variables and how to output them using the Console.WriteLine method.

C# Conditional Statements

In C#, you can use if, else if, and else statements to make decisions in your program based on conditions.

using System;

class Program
{
    static void Main(string[] args)
    {
        int age = 18;

        if (age >= 18)
        {
            Console.WriteLine("You are an adult.");
        }
        else
        {
            Console.WriteLine("You are a minor.");
        }
    }
}

This program checks if the user is an adult (18 or older) and prints the appropriate message.

C# Loops

Loops allow you to execute a block of code multiple times. C# supports several types of loops, including for, while, and foreach.

For Loop Example

using System;

class Program
{
    static void Main(string[] args)
    {
        for (int i = 0; i < 5; i++)
        {
            Console.WriteLine(i);
        }
    }
}

This for loop prints the numbers from 0 to 4.

While Loop Example

using System;

class Program
{
    static void Main(string[] args)
    {
        int i = 0;
        while (i < 5)
        {
            Console.WriteLine(i);
            i++;
        }
    }
}

This while loop also prints the numbers from 0 to 4.

C# Arrays

Arrays in C# are used to store multiple values in a single variable. Here's an example of how to declare and use arrays:

using System;

class Program
{
    static void Main(string[] args)
    {
        int[] numbers = { 1, 2, 3, 4, 5 };

        foreach (int num in numbers)
        {
            Console.WriteLine(num);
        }
    }
}

This program uses a foreach loop to print all elements of the numbers array.

C# Functions

Functions (or methods) in C# are used to group reusable code into a single unit. Functions can take parameters and return values.

using System;

class Program
{
    static void Main(string[] args)
    {
        Console.WriteLine(Sum(5, 3));
    }

    static int Sum(int a, int b)
    {
        return a + b;
    }
}

This program defines a function called Sum that takes two integers as parameters and returns their sum.
