Basic Python Knowledge

Introduction to Python

Python is a high-level, interpreted programming language known for its simplicity and readability. It is widely used for web development, data science, automation, and artificial intelligence.

Variables and Data Types

Variables in Python are used to store data. Python has several data types, including:

  • Integers: Whole numbers (e.g., 10, 42)
  • Floats: Numbers with decimal points (e.g., 3.14, 0.5)
  • Strings: Text (e.g., "Hello", 'Python')
  • Booleans: True or False

Here's an example of defining variables:

# Example of defining variables
name = "Alice"    # String
age = 25          # Integer
height = 5.6      # Float
is_student = True # Boolean

Control Flow: Conditional Statements

Conditional statements allow us to control the flow of our program based on certain conditions. The most commonly used conditionals are if, elif, and else.

# Example of if-elif-else
age = 18

if age < 18:
    print("You are a minor.")
elif age == 18:
    print("You are 18 years old.")
else:
    print("You are an adult.")

Loops: Repeating Actions

Loops allow us to repeat actions multiple times. Python has two main types of loops: for loops and while loops.

For Loop Example

# Example of a for loop
for i in range(1, 6):
    print(i)  # Output: 1, 2, 3, 4, 5

While Loop Example

# Example of a while loop
count = 1
while count <= 5:
    print(count)  # Output: 1, 2, 3, 4, 5
    count += 1

Functions

A function is a block of code that only runs when it is called. It can take input (called parameters) and return an output.

# Example of a function
def greet(name):
    return "Hello, " + name

print(greet("Alice"))  # Output: Hello, Alice

Conclusion

Python is a versatile and beginner-friendly language. With the basics of variables, data types, control flow, loops, and functions, you're ready to start writing your own Python programs!
