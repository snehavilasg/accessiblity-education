CSS Basics for Learners

Introduction to CSS

CSS (Cascading Style Sheets) is a language used to describe the presentation of a document written in HTML. It controls the layout of multiple web pages in a website. CSS allows you to style the elements on your webpage, such as setting the colors, fonts, margins, and spacing.

CSS Syntax

CSS rules are made up of selectors and declarations. A CSS rule consists of a selector followed by a declaration block. The declaration block contains one or more declarations, each made up of a property and a value.

/* CSS Rule Syntax */
selector {
    property: value;
}

/* Example: */
h1 {
    color: blue;
}

In the example above, the h1 selector is targeted, and the color property is set to blue.

CSS Selectors

CSS selectors are used to select the HTML elements that you want to style. There are different types of selectors:

  • Element Selector: Targets HTML elements by their tag name.
  • Class Selector: Targets elements with a specific class attribute (prefixed with .).
  • ID Selector: Targets an element with a specific ID attribute (prefixed with #).
  • Universal Selector: Targets all elements on the page (using *).

Example:

/* Element selector */
h2 {
    color: green;
}

/* Class selector */
.button {
    background-color: red;
}

/* ID selector */
#header {
    font-size: 2rem;
}

In the above example:

  • The h2 element will be styled with green color.
  • Any element with the button class will have a red background.
  • The element with the id="header" will have a font size of 2rem.

Colors and Backgrounds

CSS allows you to set colors and backgrounds for your webpage elements. You can use named colors, hex color codes, RGB, or HSL values.

Example of Setting Background Color:

/* Using a color name */
body {
    background-color: lightblue;
}

/* Using a hex code */
h1 {
    color: #ff6347; /* Tomato */
}

/* Using RGB */
p {
    color: rgb(0, 128, 0); /* Green */
}

In this example, we set the background color of the body element to light blue, the h1 color to tomato, and the p color to green using RGB.

Text Styling

CSS provides several properties for styling text, such as changing the font, size, weight, and style. Here are a few examples:

/* Font properties */
p {
    font-family: 'Arial', sans-serif;
    font-size: 1.2rem;
    font-weight: bold;
}

/* Text properties */
h3 {
    text-align: center;
    text-decoration: underline;
}

In this example:

  • The p elements have their font set to Arial, with a size of 1.2rem, and bold text.
  • The h3 elements are centered and have underlined text.

Layout and Positioning

CSS allows you to control the layout and positioning of elements. Some of the key properties include:

  • margin: Controls the space outside an element.
  • padding: Controls the space inside an element.
  • display: Specifies how an element is displayed (block, inline, flex, grid, etc.).
  • position: Specifies how an element is positioned (static, relative, absolute, fixed).

Example:

/* Margin and Padding */
div {
    margin: 20px;
    padding: 10px;
}

/* Positioning */
#box {
    position: relative;
    top: 50px;
    left: 30px;
}

In this example:

  • The div elements have a margin of 20px and padding of 10px.
  • The element with id="box" is positioned 50px from the top and 30px from the left of its normal position.
