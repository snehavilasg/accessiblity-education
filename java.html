Introduction to Java

Java is a popular, high-level programming language known for its platform independence, meaning Java programs can run on any system with a Java Virtual Machine (JVM) installed. It is widely used for building applications, web servers, mobile apps, and more. This page introduces basic operations in Java.

Basic Java Operations

In Java, you can perform arithmetic, comparison, and logical operations. Here are examples:

/* Arithmetic Operations */
int x = 10;
int y = 5;
System.out.println(x + y); // Output: 15
System.out.println(x - y); // Output: 5
System.out.println(x * y); // Output: 50
System.out.println(x / y); // Output: 2
System.out.println(x % y); // Output: 0

/* Comparison Operations */
System.out.println(x == y);  // Output: false
System.out.println(x != y);  // Output: true
System.out.println(x > y);   // Output: true
System.out.println(x < y);   // Output: false

/* Logical Operations */
System.out.println(x > 5 && y < 10);  // Output: true
System.out.println(x > 5 || y < 3);   // Output: true
System.out.println(!(x > 5));          // Output: false

Variables in Java

In Java, variables are containers for storing data values. Java is a statically typed language, meaning you need to declare the type of a variable before using it. Here are examples of variables:

/* Declaring Variables */
int age = 25; // Integer variable
double height = 5.9; // Double variable
char grade = 'A'; // Character variable
boolean isJavaFun = true; // Boolean variable

/* Printing Variables */
System.out.println("Age: " + age);         // Output: Age: 25
System.out.println("Height: " + height);   // Output: Height: 5.9
System.out.println("Grade: " + grade);     // Output: Grade: A
System.out.println("Is Java Fun: " + isJavaFun); // Output: Is Java Fun: true

Loops in Java

Loops in Java are used to execute a block of code repeatedly based on a condition. Java supports several types of loops, including for, while, and do-while loops.

/* For Loop Example */
for (int i = 0; i < 5; i++) {
    System.out.println("i = " + i); // Output: i = 0, 1, 2, 3, 4
}

/* While Loop Example */
int j = 0;
while (j < 5) {
    System.out.println("j = " + j); // Output: j = 0, 1, 2, 3, 4
    j++;
}

/* Do-While Loop Example */
int k = 0;
do {
    System.out.println("k = " + k); // Output: k = 0, 1, 2, 3, 4
    k++;
} while (k < 5);

Methods in Java

A method in Java is a block of code that performs a specific task. It is used to define reusable pieces of logic. Below is an example of how to create and call methods in Java:

/* Method Definition */
public class Main {
    // A method that adds two numbers
    public static int addNumbers(int a, int b) {
        return a + b;
    }

    public static void main(String[] args) {
        // Calling the addNumbers method
        int result = addNumbers(10, 20);
        System.out.println("Sum: " + result); // Output: Sum: 30
    }
}
