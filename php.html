PHP Basics for Learners

What is PHP?

PHP is a popular server-side scripting language used for web development. It is commonly embedded in HTML to create dynamic content, interact with databases, and handle user input on web pages.

PHP Syntax

PHP scripts start with <?php and end with ?>. You can mix HTML and PHP within a single file. PHP code is executed on the server, and the result is sent to the browser as plain HTML.

<?php
    echo "Hello, world!";
?>

The code above outputs "Hello, world!" to the browser.

PHP Variables and Data Types

In PHP, variables are prefixed with a $ sign. PHP supports several data types such as strings, integers, floats, booleans, arrays, etc.

<?php
$name = "John";       // String
$age = 25;            // Integer
$isStudent = true;    // Boolean
$height = 5.9;        // Float

echo "Name: " . $name;
echo "Age: " . $age;
echo "Student: " . $isStudent;
?>

This code demonstrates variable declaration and outputs the values using echo.

PHP Conditionals

PHP allows you to use if, else, and elseif statements to control the flow of your program.

<?php
$age = 18;

if ($age >= 18) {
    echo "You are an adult.";
} else {
    echo "You are a minor.";
}
?>

This example checks if a user is an adult or a minor based on their age.

PHP Loops

Loops allow you to execute a block of code multiple times. PHP supports for, while, and foreach loops.

For Loop Example

<?php
for ($i = 0; $i < 5; $i++) {
    echo $i . "<br>";
}
?>

This for loop prints numbers from 0 to 4.

While Loop Example

<?php
$i = 0;
while ($i < 5) {
    echo $i . "<br>";
    $i++;
}
?>

This while loop also prints numbers from 0 to 4.

PHP Arrays

Arrays in PHP are used to store multiple values in a single variable. There are indexed arrays and associative arrays.

Indexed Array Example

<?php
$fruits = array("Apple", "Banana", "Orange");
echo $fruits[0]; // Outputs "Apple"
?>

Associative Array Example

<?php
$person = array("name" => "John", "age" => 25, "city" => "New York");
echo $person["name"]; // Outputs "John"
?>

PHP Functions

Functions allow you to reuse code. You can define your own functions in PHP using the function keyword.

<?php
function greet($name) {
    return "Hello, " . $name;
}

echo greet("John"); // Outputs "Hello, John"
?>

This example defines a function greet() that takes a parameter and returns a greeting message.
